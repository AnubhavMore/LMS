# PRODUCT BRIEF: Edge Learning & Delivery OS (Edge LMS 2030)

## Core Product Stance
- **Not a Demo:** This is a production-grade web application for Strengthscape.
- **Code-First:** Built from scratch using modern web technologies (Next.js/React), completely replacing the initial Bubble clone prototype.
- **Enterprise-Led:** The UI and workflows should feel premium, calm, and consulting-led, never flashy or overwhelming.

## Primary Purpose
Edge LMS 2030 is a multi-tenant learning, delivery, reporting, and client-retention system connected conceptually to the Edge10 commercial operating system. It moves beyond standard video-delivery LMS platforms by integrating AI-guided insights, assessment interpretation, and real-world action planning for organizations.

## Multi-Tenant Foundation
- Multi-tenancy is the core architectural principle from day one.
- Every client company must be strictly isolated. A user from Company A must never be able to see data, courses, or insights from Company B under any circumstances.

## Target Roles & Personas
The system serves distinct roles, each with a specialized dashboard and permissions:
1. **Platform Super Admin**: Full platform access (Strengthscape IT/Leadership).
2. **Strengthscape Admin**: Internal operations and management.
3. **Content Admin**: Course creators and asset managers.
4. **Consultant/Facilitator**: Trainers delivering live cohorts and managing session feedback.
5. **Client Admin**: HR/L&D buyers managing their own company's learning journey and viewing insights.
6. **Client Manager**: Line managers monitoring their direct team's progress.
7. **Learner**: End-users consuming courses, assessments, and taking quizzes.
8. **Finance/Ops**: Managers monitoring subscription states and billing.

## AI Philosophy
- **Quota-Controlled:** Live AI token usage must be minimized and aggressively managed.
- **No Open-Ended Chatbots:** In Phase 1, there is no generic chatbot. 
- **Structured AI:** Intelligent behavior will emerge from templates, rule tables, seeded recommendation libraries, and structured program data.
- **Admin-Triggered:** Optional AI execution is admin-only, manually triggered, and logged.

## Summary
Edge LMS will act as the single source of truth for Strengthscape’s programmatic learning delivery, providing clients with unparalleled visibility into engagement, retention, and behavioral change through guided insights.
