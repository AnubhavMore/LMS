# PHASE PLAN: Edge LMS 2030

## Locked Scope: Phase 1 (P1 Priorities)
The focus of Phase 1 is laying a rock-solid, secure, multi-tenant foundation, delivering core learning pathways, and enabling administrative control.

1. **Sprint 0: Foundation Setup**
   - Data types, Option Sets, Privacy rules (Tenant Isolation), UI System, and Folder Structure.
2. **Sprint 1: Login & Users**
   - Authentication, Invite flow, Company/Tenant setup, Role assignment, Dashboard skeletons.
3. **Sprint 2: Core Learning Engine**
   - Courses, Modules, Lessons, Vimeo video embedding, Course Player UI, Progress tracking.
4. **Sprint 3: Quizzes & Certificates**
   - Quiz pass/fail logic, Completion logic, Certificate generation.
5. **Sprint 4: Payments & Access**
   - Razorpay integration, Plan records, Provisioning automation via Webhooks.
6. **Sprint 5: Admin Tools**
   - Learner management, Course assignment, Basic reporting (CSV export), Automated emails.

## Phase 2 Overview (P2 Priorities)
- **AI-Guided Insights:** Structured 10-question engine yielding consulting-grade answers.
- **Heat Maps & Advanced Reports:** Pre-computed completion matrices by department.
- **Assessment File Upload:** AI interpretation of diagnostics.
- **AI Content Engine:** Internal tool for generating lesson summaries and nudges (admin-approved).

## Phase 3 Overview (P3 Priorities)
- **Consultant Workspace:** Notes and action tracking for active clients.
- **Culture Journey Analytics:** Long-term timeline and milestone mapping.
- **Integrations:** Zoho CRM writeback and Microsoft 365 sync.
